# Warmingtea — Web

Фронтенд (Next.js / React) приложения Warmingtea.

Общее описание проекта, архитектура, переменные окружения и инструкции по запуску — в [корневом README](../../README.md).

## Быстрый старт

```bash
pnpm dev:web   # из корня монорепозитория, http://localhost:3000
```

## Тесты

```bash
pnpm test        # одноразовый прогон (Vitest)
pnpm test:watch  # ре-ран при изменениях
pnpm test:ui     # web-UI с подробностями
```
