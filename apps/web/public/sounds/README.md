# Звуки

Положите сюда `reminder.mp3` — короткий звук foreground-напоминания (проигрывается,
когда вкладка открыта и приходит push). Файл предоставляет заказчик.

Если файла нет — напоминание всё равно работает: фоновый системный баннер со
стандартным звуком (когда сайт закрыт) и in-app тост + вибрация (когда открыт),
просто без кастомного звука (`Audio.play()` тихо падает).
